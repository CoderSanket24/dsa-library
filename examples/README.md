# Real-World DSA Problem Solutions

This directory contains practical examples demonstrating how to use the DSA library to solve real-world programming problems.

## Problems Included

1. **Expression Evaluation** - Using Stack to evaluate postfix expressions
2. **Task Scheduler** - Using Queue for job scheduling
3. **Social Network** - Using Graph for friend connections
4. **Student Database** - Using BST for efficient searching
5. **Palindrome Checker** - Using Stack for string validation
6. **Level Order Traversal** - Using Queue for tree traversal
7. **Top K Elements** - Using sorting algorithms
8. **Duplicate Detection** - Using searching algorithms
9. **Undo/Redo System** - Using Stack for operation history
10. **Shortest Path** - Using Graph and Queue (BFS)

## Building Examples

```bash
# Build all examples
make examples

# Build individual example
make example_expression
make example_scheduler
make example_social
# ... etc
```

## Running Examples

```bash
./example_expression
./example_scheduler
./example_social
# ... etc
```
